# Library System

**Library System** adalah platform peminjaman buku berbasis web yang dikembangkan untuk memudahkan pengguna dalam mencari, meminjam, dan membaca buku secara modern dan efisien. Sistem ini dirancang sebagai solusi atas keterbatasan perpustakaan konvensional yang masih bergantung pada proses manual, sehingga meningkatkan aksesibilitas dan pengalaman pengguna.

Pengembangan sistem ini sejalan dengan **Sustainable Development Goals (SDGs)**, khususnya **SDG 4: Pendidikan Berkualitas**, yang menekankan pentingnya akses pendidikan yang inklusif, merata, dan berkelanjutan bagi seluruh lapisan masyarakat.

## Fitur Utama:
- Antarmuka website yang **intuitif dan responsif**
- **Real-time update** ketersediaan buku
- Akses langsung ke **E-Book**
- **Localization**: pengguna dapat memilih bahasa **Indonesia** atau **English** pada halaman Login dan Register
- Pencarian buku berdasarkan **judul**
  
## Page:
### Main Page
- **Homepage:**
  <br/>Menampilkan beberapa buku yang diunggulkan dalam sistem
- **View All Books:**
  <br/>Menampilkan seluruh buku dalam sistem dan difilter berdasarkan **genre** buku
- **About Us**
  </br>Menjelaskan tujuan dan latar belakang website library system
### User Only Page
- **Borrowed Books :**
  - Melihat daftar buku yang sedang dipinjam
  - Mengembalikan buku
  - Link untuk membaca E-Book

### Admin Only Page
- **Manage Book Page:**
  - Menambahkan buku baru
  - Menghapus buku yang sudah ada
  - Mengubah informasi buku dalam database
  
## Tech Stack Used
